# A Minimum Counterexample Proof of the Seymour Second Neighborhood Conjecture via the Graph Level Order 

The Seymour Second Neighborhood Conjecture asks if every oriented graph contains a node whose second neighborhood is at least as large as its first. This repository shows through both JavaScript and Python that my algorithm can solve the conjecture for graphs of any size. 

## Installation

Files are simply browser dependent. 

## Usage

JavaScript 
1. Every finite graph has a minimum out-degree node. That is the input into the file 'dgsquare.htm'
2. Upon entering this input, the algorithm will execute, displaying decreasing sequences of neighborhoods until one with Seymour vertices is forced by the binomial theorem.
